# web-scraper

🌐 **English** | [日本語](#日本語)

> A general-purpose web data extraction tool generated by the **prompt-driven method** described in this Zenn article: [https://zenn.dev/nobmake/articles/4958a180f651ca](https://zenn.dev/nobmake/articles/4958a180f651ca).

This branch contains two deliverables that were produced using the prompt technique from the article above, applied to [**Nous Research Hermes**](https://github.com/NousResearch/hermes-agent) — an autonomous AI agent runtime — to give it a brand-new `web_scraper_extract` capability.

| Deliverable | What it is |
|---|---|
| 📦 [`web_scraper_tool/`](web_scraper_tool/) | A standalone Python package. Given a URL and a JSON Schema (or a natural-language description), it returns structured JSON via a 3-stage pipeline (static fetch → dynamic fetch with Playwright → LLM-driven extraction). Built-in **SSRF defense**, **robots.txt respect**, **prompt-injection-resistant** logging, secret redaction, and an **OpenAI-compatible LLM backend** (defaults to a vLLM endpoint, swappable). |
| 🔌 [`plugins/web_scraper/`](plugins/web_scraper/) | A thin **Hermes plugin** adapter that wraps the package above and exposes it to a Hermes agent runtime as a tool named `web_scraper_extract` (toolset: `web_scraper`). |

> Part of the [`vibe-whims`](https://github.com/nob-git-dev/vibe-whims) experiment repository.

## Quick start

### Use the standalone library

```bash
cd web_scraper_tool/
uv venv
uv pip install -r requirements.txt
uv run playwright install chromium   # only if you need dynamic rendering

# Run the test suite (no network needed; all I/O is mocked)
uv run pytest -v
```

Then from Python:

```python
from web_scraper_tool import extract_web_data
import json

raw = extract_web_data(
    url="https://example.com/article",
    extraction_schema=json.dumps({
        "type": "object",
        "required": ["title"],
        "properties": {
            "title":  {"type": "string"},
            "author": {"type": "string"},
        },
    }),
)
result = json.loads(raw)
if result["success"]:
    print(result["data"])
else:
    print(result["error"]["stage"], result["error"]["message"])
```

`extract_web_data` **never raises**. It always returns a JSON string with `success`, `data`, `error`, `metadata`. See [`web_scraper_tool/README.md`](web_scraper_tool/README.md) for the full surface area, the 7 well-defined `error.stage` values, sample outputs, and security warnings.

### Run the plugin tests

```bash
cd plugins/web_scraper/
uv venv
uv pip install pytest pyyaml jsonschema ruff
PYTHONPATH=../.. uv run pytest tests/ -v
```

To actually wire this into a running Hermes instance, drop `plugins/web_scraper/` into your Hermes `plugins/` directory and ensure `web_scraper_tool/` is reachable on `PYTHONPATH`. See [`plugins/web_scraper/README.md`](plugins/web_scraper/README.md) for the LLM-facing schema, the return envelope, deployment notes, and known limitations.

## LLM backend

Both deliverables default to an **OpenAI-compatible** endpoint. The defaults assume a locally hosted vLLM (or compatible) server; override them with environment variables:

| Variable                       | Default                       | Notes |
|--------------------------------|-------------------------------|-------|
| `WEB_SCRAPER_LLM_MODEL`        | `qwen3.5-122b`                | Any chat-completion model the backend knows. |
| `WEB_SCRAPER_LLM_BASE_URL`     | `http://localhost:8000/v1`    | Any OpenAI-compatible base URL. |
| `WEB_SCRAPER_LLM_API_KEY`      | `EMPTY`                       | Optional. vLLM accepts anything. |
| `WEB_SCRAPER_LLM_ENABLE_THINKING` | `false`                    | Set to `true` for reasoning-style models. |

A template is provided as [`web_scraper_tool/dotenv-example.txt`](web_scraper_tool/dotenv-example.txt).

### LLM バックエンド

両成果物とも **OpenAI 互換** エンドポイントを既定としています。既定値はローカル vLLM（または互換サーバ）を想定しており、環境変数で差し替えられます:

| 変数 | 既定値 | 説明 |
|---|---|---|
| `WEB_SCRAPER_LLM_MODEL` | `qwen3.5-122b` | バックエンドが知っている任意の chat-completion モデル。 |
| `WEB_SCRAPER_LLM_BASE_URL` | `http://localhost:8000/v1` | 任意の OpenAI 互換ベース URL。 |
| `WEB_SCRAPER_LLM_API_KEY` | `EMPTY` | 任意。vLLM は無認証で受け取る。 |
| `WEB_SCRAPER_LLM_ENABLE_THINKING` | `false` | reasoning スタイルのモデルを使うときに `true`。 |

雛形は [`web_scraper_tool/dotenv-example.txt`](web_scraper_tool/dotenv-example.txt) に同梱しています。
